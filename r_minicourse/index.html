R mini-course at NORC, Academic Research Centers

welcome!

The slides and notes links on the left will be populated with content as the course progresses. If a link is dead, then that means I haven't posted the corresponding item yet. It'll be there eventually. Stay tuned.

Below is a rough outline of the topics we'll cover in the class. It's subject to change if there's a critical mass of people who want to focus on some particular aspect. I've also collected some extremely helpful cheatsheets and posted them below. There's also an ever-growing list of links to interesting blog posts, tutorials, articles, etc. about using/learning R, about computing in general, about reproducibility in science, and related topics. I'll try to select technical ones that don't require too much background beyond what we'll cover in the class.



week 1: R programming + workflow fundamentals

meta topics:

- ide's and gui's +c
- directories and files (incl wd)
- working with files, doc types
- project dir structures
- the command line
- keyboard shortcuts

first steps:

- variables + assignments
- data types 
- functions + arguments
- assignments + calls + control flow (oh my!)
- comments comments comments!
- errors and warnings

second steps:

- packages (and libraries)
- data structures (esp df)
- control flow
- coding style
- more than one way to skin a cat


week 2: working with data

file i/o:

- reading data
- get data from the web
- data file formats
- writing data

data manipulation:

- subsetting
- joining/merging
- cleaning/munging 
- dplyr
- aggregation

optimizing workflow:

- writing functions
- regex
- modifying w lookup tables
- pipechains via magrittr
- the concept of tidy data


week 3: plotting/vizualizing data

before plotting:

- summarizing data
- screening
- reshaping

making plots:

- base graphics
- ggplot2
- "reproducible" plots
- embedding plots (segue to week 4)


week 4: Rmarkdown and working with the web

making nice clean documents really easily:

- (r)markdown
- plain-text workflow
- calling stuff from other scripts

getting help and/or sharing your work:

- format for reprex/mwe
- managing/listing dependencies
- reproducibility and 21st century science

cheatsheets
Here's a handful of nice cheatsheets for various aspects of the R and R Studio ecosystem.
  • R Studio IDE cheatsheet -- quick reference for common stuff in R Studio, also a bit on keyboard shortcuts (by R Studio)
  • base R cheatsheet -- the very basics of many frequent topics (by Mhairi McNeill)
  • data import cheatsheet -- for reading in data from various formats/filetypes (by R Studio)
  • data wrangling cheatsheet -- this is probably the most important one. uses dplyr:: workflow (by R Studio)
  • data vizualization cheatsheet -- very useful for helping make plots, but requires some knowledge of how ggplot2:: works (by R Studio)
  • R Markdown cheatsheet -- most of what you'll need to make nice pretty documents very easily! (by R Studio)
You can also download a zip archive that contains all of these plus some other ones that you'll definitely want to have the deeper and deeper you get into R.
